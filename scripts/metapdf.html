Using the pdftk (PDF toolkit) one can change the metadata on non-copy-protected PDFs. I've written two scripts to help make this easier.

pdfauth reads and sets the Author field in the PDF metadata.
Usage:
   pdfauth FILE.pdf
outputs the author info, and
   pdfauth FILE.pdf "Author name"
set the metadata.

pdftit reads and sets the Title field in the PDF metadata.
Usage:
   pdftit FILE.pdf
outputs the title info, and
   pdftit FILE.pdf "PDF title"
set the metadata.
