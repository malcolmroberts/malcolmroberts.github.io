Patches for Intel OpenCL for Debian

The Intel drivers for OpenCL on CPU available at intel.com were based on the RPMs and ported to .debs via alien. Unfortunately, the install scripts were Fedora-specific, so the resulting .debs would fail on install.

Based on the comments in https://software.intel.com/en-us/forums/topic/515680, I created patches and a script to apply the patches so that one could install.

The patches are available on github and the resulting .deb files are available here:

  • opencl-1.2-base-4.4.0.117-1.x86_64.patched.deb
  • opencl-1.2-devel-4.4.0.117-1.x86_64.patched.deb
  • opencl-1.2-intel-cpu-4.4.0.117-1.x86_64.patched.deb
  • opencl-1.2-intel-devel-4.4.0.117-1.x86_64.patched.deb
